# ChessMe

ChessMe is a clean, simple iOS chess app built with SwiftUI. The goal is to create a focused chess experience for players who want to play, learn, and improve one move at a time.

## Overview

ChessMe starts as a local two-player chess game for iPhone. Players can play white and black on the same device with a clean board, smooth interactions, legal move validation, and basic game state tracking.

Future versions may include AI opponents, move history, timers, lessons, puzzles, and online multiplayer.

## Core Features

- Native iOS app built with SwiftUI
- Local two-player chess gameplay
- Interactive chessboard
- Standard chess piece movement
- Legal move validation
- Check, checkmate, and stalemate detection
- Captured pieces tracking
- New game / restart option
- Clean and minimal interface

## MVP Goals

The first version of ChessMe should focus on:

1. A polished chessboard UI
2. Smooth piece selection and movement
3. Correct chess rules
4. Local player-vs-player mode
5. Clear game status messages
6. Stable, App Store-ready foundation
